Description

Mighty Bits is a free and open source software which enables people without any background in programming to develop their own database system. This open data platform enables citizens, civil society groups and other types of organizations to share data with other communities.

Visual Programming

Export working website for data input from visual model

Data Visualization

Visualize your data with Data Visualization plugin. More plugins are planned for future version

Open Data

With easy to understand visual model make it easier to share your database platform to other communities or within team.
